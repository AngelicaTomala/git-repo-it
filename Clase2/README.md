# Clase 02 - GIT

## Para visualizar ai hay cambios

```bash
git status
```

## agregar todos los cambios en el working directory

```bash
git add .
```

## hago commits de los cambios

```bash
git commit -m "Agrego carpeta clase2 con README.md"
```

## con el git commit solo me abre el editor
```bash
git commit
```
